Texto antes do botão

Aqui está o texto que pode ser mostrado ou escondido.

Pos divv

Neste exemplo, o CSS define o elemento com o id "texto-escondido" com display:

none; para que ele esteja inicialmente oculto. O JavaScript define a função toggleTexto() que é chamada quando o botão é clicado.

Essa função verifica o estado de exibição atual do elemento e alterna entre "block" (exibido) e "none" (oculto).

Ao clicar no botão, o texto dentro da

será mostrado ou escondido, dependendo do seu estado atual.
